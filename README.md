# TrueReplayer

A fast, lightweight **macro recorder for Windows**. Record your mouse clicks, key presses, and scrolls — then replay them any time, on demand or with a hotkey.

![Version](https://img.shields.io/badge/version-2.6.0-6bcb77?style=flat-square)

## What it does

- **Record & replay** — capture exactly what you do and play it back perfectly, as many times as you want.
- **Hotkeys** — trigger any macro with a single key, even while another app is in focus.
- **Profiles & folders** — save your macros, organize them into colored folders, and bind each one to a specific window.
- **Auto-clicker** — a dedicated mode for fast, steady clicking, with adjustable speed and randomized jitter.
- **Smart steps** — go beyond plain recording: add pauses, wait for an image or a pixel color, type text, run another macro, or branch with IF conditions.
- **Works in games** — sends real input that games actually recognize (tested in Roblox and others).

## Quick start

1. Click **Recording**, do your actions, then click **Recording** again to stop.
2. Click **Replay** to play it back.
3. Click **Save**, give it a name, and assign a **Hotkey** to trigger it any time.

That's all you need to get going.
