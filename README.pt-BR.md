# TrueReplayer

Gravador e reprodutor de macros para Windows — rápido e leve. Grave cliques, teclas e scrolls e reproduza sob demanda ou por hotkey. Suporta busca por imagem, automação de navegador, encadeamento de profiles e modo auto-clicker dedicado.

## Por que usar o TrueReplayer

- **Grave uma vez, reproduza em qualquer lugar** — captura input real via low-level hooks do Windows
- **Baseado em profiles** — organize macros em pastas coloridas, atribua hotkeys, vincule a janelas específicas
- **Vai além da gravação** — adicione pauses, esperas por imagem, sub-macros e ações de navegador manualmente
- **Funciona em jogos** — `SendInput` com compatibilidade Raw Input (Roblox, jogos Unity, etc.)
- **Anti-detecção** — jitter aleatório em delays e posição dos cliques

## Instalação

Baixe a versão mais recente em [TrueReplayer-releases](https://github.com/fatalihue/TrueReplayer-releases/releases/latest):

- `TrueReplayer-win-Setup.exe` — instalador com auto-update integrado
- `TrueReplayer-win-Portable.zip` — versão portátil, sem instalação

Windows 10/11, x64.

## Começo rápido

1. Clique em **Recording**, execute as ações no app desejado, clique em **Recording** de novo para parar
2. Clique em **Replay** para reproduzir
3. Clique em **Save** para salvar o profile e atribua uma **Hotkey** para disparar com uma tecla

Pronto. Para tudo mais — espera por imagem, sub-macros, vínculo com janela, hotstrings, auto-clicker — veja o tutorial abaixo.
