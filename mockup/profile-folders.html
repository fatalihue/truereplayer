Behavior Notes

Pin to Top

Right-click any profile → Pin to Top.
Pinned profiles appear in a dedicated section at the top, separated by a divider.
Drag to reorder within pinned section. Click pin icon to unpin.

Pin

Folders

Create folders via the +Folder button in the header or right-click → Move to Folder...

Folders are virtual — they don't change the file system. Order is stored in profile-order.json.

Each folder has a customizable color (right-click folder → Change Color).

Click chevron to collapse/expand. Collapsed folders show item count.

Folder

Drag & Drop

Drag profiles via the grip handle (left dots) to:

• Reorder within the same section
• Move between folders (drop on folder header)
• Move to/from pinned section
• Move to Ungrouped (drop below all folders)

Blue drop indicator shows insertion point. Folder highlights on hover.

Drag

Storage

Profile order, pins, and folder assignments stored in:
Profiles/profile-order.json

Format: {"pinned": [...], "folders": {"name": {"color": "#60CDFF", "items": [...]}}, "order": [...]}

Profile files on disk are not moved. Folders are metadata only.
